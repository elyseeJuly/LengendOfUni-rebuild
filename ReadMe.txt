感谢您下载“刘慈欣群星传”的源代码。为了能正确编译“刘慈欣群星传”，请安装VC2005和directx9.0 sdk(片头动画所需，若不编译片头动画则不用装)，推荐在windows xp sp3下进行编译。

整个代码分为三个模块：
1、LegendOfUni : 刘慈欣群星传主程序
2、3DPrelude : 3D片头
3、MusicPlayer : 音乐播放

编译后的程序会生成在 RunEnv 目录下。为减少下载包的大小，RunEnv里的资源文件恕不附带。请您到 http://www.ourshow2003.com/lou/index.asp 下载“刘慈欣群星传”的原版游戏，并将其拷至RunEnv下即可。
