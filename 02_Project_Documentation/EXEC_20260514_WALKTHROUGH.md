## 开发实录：P3 - P9 核心代码实现

### P3/P4: 底层机制与面壁者系统
*   **纪元推进**：`CGame` 新增 `m_epoch`，根据游戏年份自动从“危机纪元”推进至“银河纪元”，每次跨纪元均载入《地球史书》。
*   **智子锁死**：地球科技研发受到智子干扰，无核心反制科技时，研发进度强制缩减90%。
*   **面壁者/执剑人**：在 `CEarthCivilization` 中实现了 `m_vecWallfacers` 和 `m_sSwordholder`。面壁者提供稳定的防御加成，执剑人提供威慑度保护。

### P5: AI威胁与安全阀
*   在 `CAlienCivilization::RunARound` 中引入“水滴与二向箔打击冷却”，外星高级AI发起攻击前会有5-10回合强制预警，留给地球文明喘息与反制空间。

### P6: 事件链系统
*   使用 Python 脚本生成了包含“古筝行动”、“智子宣言”、“月球坠落危机”的 `gameevent.ini`。在 C++ 层硬编码了月球危机的惩罚（扣除50%人口）与化解条件。

### P7: 行星发动机与数字生命 & UI全面解耦
*   **全局UI宏替换**：运行 `refactor_enums.py` 将项目中数百个过时的 `TT_NUCLEAR` 等枚举全局重定向至新的 5大支柱科技树，彻底解耦 UI 层，消除编译报错。
*   **行星发动机系统**：当研发出“重核聚变行星发动机”后，所有星球上的“加工工厂”将改名为“行星发动机”，同时其资源转经济效率拔高 **500%**！
*   **数字永生系统**：将 `SetPopulation` 重写为虚函数并在 `CEarthCivilization` 中拦截。完成“数字生命上传”科技后，地球人口永久免受任何灾害减损。

### P8: 多元胜利条件
在回合结束判定（`CheckGameOver`）中重构了原本单一的“武力征服”结局，扩展为 4 种三体特有结局：
1.  **流浪胜利**：研发出“行星发动机”与“领航员空间站”后驶离太阳系。
2.  **数字胜利**：研发出“数字生命上传”后，在服务器中迎来数字永生。
3.  **威慑胜利**：当地球任命了执剑人，并坚持存活至公元 200 年（进入威慑纪元极盛期），达成恐怖平衡的威慑胜利。
4.  **征服胜利**：消灭所有外星势力。

### P9: 动态美学：双色主题切换
通过引入外挂配置 `ui_theme.ini`，实装了无需重编译的“暗黑/纯白”双色主题切换！
*   现在游戏根目录下会读取 `ui_theme.ini`。当配置 `Theme=Light` 时，游戏面板背景将使用现代的高亮白底与黑字。
*   当配置 `Theme=Dark` 时，将呈现极具深邃三体质感的“暗黑物质”配色。美术资源无需重制即可完美套用。
