# 通用事件生成系统设计规范 (Universal Event Engine v1.0)

**日期**: 2026-06-12
**状态**: Approved for Implementation

## 1. 设计目标与背景

传统策略游戏的事件系统本质上是“满足条件 -> 抽取事件 -> 结算奖惩 -> 归零”的单向脚本，存在“事件一次性”、“彼此孤立”以及“缺乏世界沉浸感”三大根本问题，导致游戏中后期沦为枯燥的“高阶计算器”。

**Universal Event Engine (UEE v1.0)** 的核心目标是：**让事件从“随机内容”变成“世界运行结果”**。

这套系统的核心哲学可以浓缩为：
> 状态产生行为，行为产生事件，事件留下历史，历史改变状态。

---

## 2. UEE 核心三原则

1. **第一原则：事件不是内容，而是状态的可视化**
   永远不要先设计事件，先设计世界状态。例如“港口骚乱”不是一个凭空抽取的事件，而是“治安下降、失业增加”这些状态累积后的表象结果。
2. **第二原则：所有事件必须留下痕迹**
   事件结束后决不能归零，必须在世界中沉淀出 **Tag（标签）、状态、关系或历史记录**。
3. **第三原则：所有事件都是因果链的一环**
   世界像生态系统而不是抽奖机。事件 A 导致状态变化，进而改变事件 B 的概率，解锁事件 C，封锁事件 D。

---

## 3. UEE 八层系统架构

| 层级 | 名称 | 职责与作用 |
| :--- | :--- | :--- |
| **Layer 1** | 世界状态层 (State) | 系统的核心底层。记录文明的经济、稳定度、军力、民心等纯数值。 |
| **Layer 2** | Tag 系统 (Memory) | 状态是数值，Tag 是经历的记忆。分为角色、地区、组织、历史和世界五类 Tag。 |
| **Layer 3** | 事件生成层 (Generation) | 取消纯随机抽取，采用动态公式计算权重：`EventScore = StateWeight + TagWeight + RelationWeight + TimeWeight`。 |
| **Layer 4** | 关系网络层 (Relations) | 驱动角色互动的发动机。记录盟友、宿敌、背叛等深度羁绊。 |
| **Layer 5** | 生态链系统 (Ecology) | 禁止数值下降直接跳跃到危机事件，必须经过“群体行为变化 -> 社会变化”的生态涟漪传导。 |
| **Layer 6** | 表现层 (Presentation) | 禁止直接显示数值增减，通过 UI 滤镜、背景杂音、视觉破损来反向呈现世界状态。 |
| **Layer 7** | 切片叙事 (Slice Narrative) | 将宏观的灾难具象化为小人物的日常动作和切片细节，用文学厚度包裹工程冷硬。 |
| **Layer 8** | 历史生成 (History) | 监听所有变化，自动生成世界史、年鉴或角色传记。 |

---

## 4. 《LegendOfUni》项目落地与重构规划

结合上述 UEE 架构，本项目将推进以下四个维度的引擎重构：

### 4.1 状态层与 Tag 沉淀
引入全新的 `TagManager`，核心支持 `worldTags`（如：资源枯竭边缘、科技爆发期）与 `characterTags`。摒弃“事件用完即弃”的现状，让《三体》宇宙拥有记忆。

### 4.2 表现层与切片叙事系统 (反向滤镜)
引入全局 `Atmosphere` 氛围状态机。根据 `worldTags` 或核心资源的水位线切换游戏全局滤镜（例如：处于低谷期时，UI 主色调变暗，背景加入轻微噪点）。
在事件文本上，折叠冷冰冰的数字提示，注入具象化的微观切片描写，体现小人物在大时代下的挣扎。

### 4.3 事件生成与生态链系统 (多米诺骨牌)
重写 `EventCadence.ts` 的评分引擎，将抽取机制改为状态和 Tag 导向的必然。
停止设计“一步到位”的灾难事件，拆分为多级链式反应：`[配给减少]` -> (生成 Tag: 民心不稳) -> `[切片事件：地下配给争端]` -> (生成 Tag: 地下帮派) -> `[核心危机爆发]`。

### 4.4 历史生成系统与时间线融合 (两项关键设计共识)
在架构评审中，团队明确了以下关键边界，以确保系统契合原著宏大气质：
1. **时间线即编年史**：不再单独开辟独立的“历史面板”。游戏现有的 `timeline.json` 骨架将被升级为动态编年史，动态插入玩家触发的生态事件和核心 Tag。
2. **克制的关系网络**：不盲目扩张好感度矩阵。关系网络将严格遵循《三体》原著精神，聚焦于“角色与人类社会的宏观关系”（如：为人类做贡献、脱离社会、逃亡派、人类叛徒），角色存在的核心目的是服务于文明存续的宏大命题。
