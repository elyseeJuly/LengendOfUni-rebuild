# 开放问题补充设计方案

本文档针对玩法方案中3个待定问题进行深化设计。

## 问题3：数据驱动 vs 代码驱动对比

### 对比总览

| 维度 | 数据驱动（INI配置） | 代码驱动（C++硬编码） |
|------|-------------------|---------------------|
| **修改方式** | 编辑 `.ini` 文件，无需重新编译 | 修改 `.cpp` 源码，需重新编译 |
| **开发速度** | 首次慢（需设计配置格式），迭代极快 | 首次快（直接写逻辑），迭代慢 |
| **灵活度** | 高——策划可独立调参数 | 低——每次改动需程序员介入 |
| **复杂逻辑** | 弱——INI只能表达线性数据，难以表达分支/条件 | 强——任意复杂逻辑均可实现 |
| **调试难度** | 难——配置错误只在运行时暴露，且无编译期检查 | 易——编译器帮助检查类型、语法 |
| **现有先例** | 武器/角色/星球/事件/外星文明 已全部数据驱动 | 科技树/部门逻辑/战斗算法 均为代码驱动 |

### 各新机制适用性分析

| 机制 | 推荐方案 | 理由 |
|------|---------|------|
| **面壁者系统** | **混合** | 面壁者候选人列表、属性阈值 → INI配置；面壁计划触发逻辑（秘密积累/叛变判定）→ C++代码 |
| **执剑人机制** | **混合** | 威慑度衰减速率、交接期时长 → INI配置；威慑度计算公式、三体进攻判定 → C++代码 |
| **智子封锁** | **代码驱动** | 逻辑简单但嵌入深——需修改 `CDepartment::RunARound` 中的科研减速，配置项仅一个减速百分比 |
| **黑暗森林外交** | **混合** | 6种关系状态阈值、对话文本 → INI配置；状态转换规则、AI行为决策 → C++代码 |
| **文明纪元** | **数据驱动** | 与现有 `CheckCiviLevelUpgrade` 结构一致，仅增加阈值和解锁内容，适合INI配置 |
| **AI特殊武器** | **代码驱动** | 需复杂的威胁评估和安全阀逻辑，不适合配置化 |
| **事件链** | **数据驱动** | 现有 `gameevent.ini` 已支持，仅需扩展分支选择字段 |

### 推荐结论

**采用混合方案**——与现有代码库风格一致：
- **数据层**（INI）：所有可调数值（阈值/概率/文本/人名/属性值）
- **逻辑层**（C++）：所有判定规则（状态机转换/威胁评估/连锁触发）

这样策划调数值时只需改INI，但核心逻辑的正确性有编译器保障。

## 问题4：AI文明使用特殊武器的详细设计

### AI行为总架构

现有AI逻辑在 `CAlienCivilization::RunARound()` 中，每回合执行：随机决定是否扩张→随机选星球→判定是否与玩家冲突→自动增兵/造武器。

新方案在此基础上增加 **AI决策层**，按以下优先级执行：

```
每回合AI执行流程：
  1. 威胁评估 → 计算与玩家的实力对比
  2. 战略决策 → 根据文明性格选择行动
  3. 特殊武器判定 → 是否使用终极武器
  4. 常规行动 → 扩张/增兵/造武器（现有逻辑）
```

### AI文明性格系统

每个外星文明增加 `AI_PERSONALITY` 属性（在 `alien.ini` 中配置）：

| 性格类型 | 代表文明 | 特殊武器倾向 | 行为特征 |
|---------|---------|-------------|---------|
| **猎手型** | 三体 | 水滴优先 | 主动进攻，优先消灭威胁最大的玩家星球 |
| **清理者型** | 歌者 | 二向箔/光粒优先 | 不主动接触，但一旦发现就执行清理 |
| **防御型** | 碳基联盟 | 不使用终极武器 | 只在被攻击时反击 |
| **扩张型** | 恐龙文明 | 常规武器 | 专注占领无主星球 |
| **投机型** | 波江座 | 根据形势选择 | 进攻弱者、回避强者 |

### 特殊武器AI使用规则

#### 水滴（强相互作用力探测器）

```
触发条件：
  - 三体文明等级 ≥ 3（掩体纪元）
  - 玩家拥有的军营数 ≥ 3
  - 距上次使用 ≥ 10回合冷却

AI行为：
  - 三体派出1个水滴攻击玩家最强军营
  - 战斗中水滴作为不可选择的敌方单位，攻击力极高（500），HP极高（999）
  - 反制手段：玩家已研究"强相互作用力材料"可解锁"水滴捕获"战术选项

安全阀：
  - 每局游戏最多使用3次
  - 若玩家实力已远低于三体（军力<三体的30%），不再使用（避免赶尽杀绝）
```

#### 二向箔（降维打击）

```
触发条件（极其严格）：
  - 仅"歌者"文明可使用
  - 歌者文明等级 ≥ 4
  - 玩家进入"广播纪元"后 ≥ 20回合
  - 玩家曾使用过"坐标广播"技能（暴露了自己的位置）

AI行为：
  - 歌者向被暴露坐标的星球发射二向箔
  - 给予玩家 5回合预警（"检测到未知维度扰动"事件）
  - 5回合后该星球被永久摧毁（所有建筑/军营/工厂/城市消失）

安全阀：
  - 必须有预警，玩家有时间撤离
  - 每局仅可触发1次
  - 若玩家已研究"二向箔防御学"，可选择"紧急降维适应"（保住星球但损失50%资源）
  - 绝不会在玩家仅剩1颗星球时触发（保证不会直接game over）
```

#### 光粒（恒星打击）

```
触发条件：
  - 仅"歌者"文明可使用
  - 作为剧情事件触发（非随机）——"波江座毁灭"事件链末尾
  - 玩家主动广播他人坐标后，目标文明会被光粒摧毁

AI行为：
  - 光粒不直接针对玩家——而是作为"黑暗森林打击"技能的执行后果
  - 玩家广播某文明坐标 → 3-8回合后该文明母星被光粒摧毁
  - 如果玩家被其他文明广播坐标 → 玩家收到预警 → 需启动"流浪地球计划"或"黑域发生器"

安全阀：
  - 完全事件驱动，不随机触发
  - 始终有逃生路径
```

#### 引力波广播（坐标暴露）

```
触发条件：
  - 任何文明被逼入绝境时（仅剩1颗星球 + 即将被消灭）
  - 类似"万有引力号"的末路行为

AI行为：
  - 濒死文明向全宇宙广播坐标（包括自己和攻击者）
  - 效果：所有被广播文明进入"坐标暴露"状态
  - 暴露状态：歌者文明启动清理倒计时

安全阀：
  - 可通过"黑域发生器"免疫
  - 可通过"光速飞船"逃离
  - 给予充足回合数准备
```

### AI威胁评估公式

```cpp
// 伪代码：AI每回合计算对玩家的威胁评估
int ThreatLevel = (PlayerArmy * 2 + PlayerTecLevel * 3 + PlayerStarCount * 5)
                - (MyArmy * 2 + MyTecLevel * 3 + MyStarCount * 5);

if (ThreatLevel > 100)      → 玩家威胁极高 → 考虑使用特殊武器
else if (ThreatLevel > 50)   → 威胁高 → 积极备战
else if (ThreatLevel > 0)    → 一般 → 防御为主
else if (ThreatLevel > -50)  → 我方优势 → 扩张
else                         → 我方碾压 → 常规行动（不使用特殊武器）
```

### AI使用特殊武器的总原则

> [!IMPORTANT]
> 1. **始终有预警**——玩家不会在毫无准备的情况下被终极武器击中
> 2. **始终有反制**——每种终极武器都有对应的科技可以防御或减轻
> 3. **绝不一击毙命**——当玩家仅剩最后资源时，AI不会使用不可逆武器
> 4. **频率受限**——所有终极武器有冷却时间和使用次数上限
> 5. **性格一致**——只有对应性格的文明才会使用对应武器，不会随机分配

## 问题5：流浪地球内容深化

### 流浪地球完整子系统设计

在"航天工程"科技树的基础上，流浪地球相关内容展开为完整的可玩系统，所有内容以标准科技树节点形式呈现（与现有 `CTecTree` / `CTecTreeNode` 框架完全一致）。

### 月球危机事件链（流浪地球II核心剧情）

以标准事件系统（`gameevent.ini`）实现：

```
事件链触发条件：进入"威慑纪元" + 回合数 > 30

[事件1] "月球轨道异常"
  对话：丁仪说："月球轨道出现微小偏移，原因不明。"
  效果：无直接效果，剧情铺垫
  
[事件2] "月球坠落预警"（事件1后5回合触发）
  对话：章北海说："月球正在加速接近地球！如果不采取措施，
        300年后将与地球碰撞！"
  效果：开始倒计时（30回合）
  玩家选择：
    A. 启动月球发动机计划 → 需已研究"月球发动机"科技
    B. 启动核弹轰击方案 → 消耗大量武器，月球碎裂（降低危机但产生碎片事件）
    C. 暂时搁置 → 倒计时继续

[事件3a] "月球发动机启动"（选A且有科技）
  对话：程心说："月球发动机点火成功！月球正在被推离危险轨道。"
  效果：月球危机解除，获得"月球基地"（额外矿场+工厂位置）
  后续解锁："月球推离系统"科技

[事件3b] "月球核爆"（选B）
  对话：维德说："前进！前进！不择手段地前进！引爆它！"
  效果：月球被摧毁，产生碎片雨 → 地球人口-30%，但危机解除
  后续：碎片事件持续3回合，每回合随机损坏一个星球建筑

[事件3c] "月球碰撞"（倒计时归零）
  效果：地球星球被摧毁 → 若无其他星球则游戏结束
  预防：研究"行星发动机Ⅰ型"可执行紧急地球变轨（最后5回合内可用）
```
